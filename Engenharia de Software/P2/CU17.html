CU17: Atribuir Funcionalidade ao Veículo

Identificador: CU17

Título: Atribuir Funcionalidade ao Veículo

Ator(es): Administrador

Descrição: Este caso de uso permite ao administrador atribuir uma funcionalidade específica a um veículo, como designá-lo para uma rota específica ou configurá-lo para um tipo de viagem (transporte de passageiros, carga, etc.).

Pré-condições

  • O administrador deve estar autenticado no sistema.

Pós-condições

  • O sistema atualiza o veículo com a funcionalidade atribuída.

Fluxo de Eventos

  1. O administrador acessa a página de gerenciamento de veículos.
  2. O administrador seleciona o veículo a ser configurado.
  3. O administrador escolhe a funcionalidade a ser atribuída (ex: transportar passageiros ou carga).
  4. O sistema valida e confirma a atribuição.
  5. O sistema atualiza o banco de dados com a nova configuração.

Requisitos Especiais

  • O sistema deve garantir que apenas administradores possam acessar essa funcionalidade.

Cenários Alternativos

  • Se a funcionalidade atribuída não for válida para o veículo, o sistema exibe uma mensagem de erro.
