CU24: Enviar Notificação de Alteração

Identificador: CU24

Título: Enviar Notificação de Alteração

Ator(es): Administrador, Passageiro

Descrição: Este caso de uso permite ao sistema enviar uma notificação aos passageiros e administradores sobre alterações em viagens, como mudanças de horário ou cancelamentos.

Pré-condições

  • A alteração na viagem deve ter sido realizada no sistema.

Pós-condições

  • O sistema envia notificações aos passageiros ou administradores envolvidos.

Fluxo de Eventos

  1. O administrador faz alterações em uma viagem, como mudança de horário ou cancelamento.
  2. O sistema gera notificações sobre as alterações.
  3. O sistema envia notificações via email, SMS ou aplicativo, dependendo da configuração do usuário.

Requisitos Especiais

  • O sistema deve garantir que as notificações sejam enviadas corretamente, para os contatos cadastrados dos passageiros e administradores.

Cenários Alternativos

  • Se a notificação não puder ser enviada, o sistema exibe uma mensagem de erro.
