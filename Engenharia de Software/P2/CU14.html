CU14: Visualizar Notificações

Identificador: CU14

Título: Visualizar Notificações

Ator(es): Passageiro

Descrição: Este caso de uso permite ao passageiro visualizar notificações relacionadas a suas viagens, como alterações de horário, cancelamentos ou promoções.

Pré-condições

  • O passageiro deve estar autenticado no sistema.

Pós-condições

  • O sistema exibe a lista de notificações disponíveis.

Fluxo de Eventos

  1. O passageiro acessa a página de notificações.
  2. O sistema exibe uma lista de notificações.
  3. O passageiro pode visualizar os detalhes de cada notificação.

Requisitos Especiais

  • O sistema deve ser capaz de enviar notificações em tempo real.

Cenários Alternativos

  • Se não houver notificações disponíveis, o sistema exibe uma mensagem informando que não há novas notificações.
