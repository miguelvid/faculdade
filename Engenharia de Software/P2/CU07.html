CU07: Alterar Dados do Passageiro

Identificador: CU07

Título: Alterar Dados do Passageiro

Ator(es): Passageiro

Descrição: Este caso de uso permite ao passageiro alterar seus dados pessoais, como nome, e-mail e telefone.

Pré-condições

  • O passageiro deve estar autenticado no sistema.

Pós-condições

  • Os dados pessoais do passageiro são atualizados no sistema.

Fluxo de Eventos

  1. O passageiro acessa a página "Meus Dados".
  2. O passageiro altera os dados desejados.
  3. O sistema valida as alterações.
  4. O sistema atualiza os dados no banco de dados.

Requisitos Especiais

  • O sistema deve garantir que as alterações sejam validadas corretamente.
  • O sistema deve enviar uma confirmação da alteração para o e-mail do passageiro.

Cenários Alternativos

  • Se os dados inseridos forem inválidos (por exemplo, um número de telefone mal formatado), o sistema exibe uma mensagem de erro.
