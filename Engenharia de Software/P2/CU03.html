CU03: Consultar Viagens Disponíveis

Identificador: CU03

Título: Consultar Viagens Disponíveis

Ator(es): Passageiro

Descrição: Este caso de uso permite ao passageiro consultar as viagens disponíveis, filtrando por diferentes critérios como destino, data e horário.

Pré-condições

  • O passageiro deve estar logado no sistema.

Pós-condições

  • O sistema exibe uma lista de viagens de acordo com os filtros aplicados.

Fluxo de Eventos

  1. O passageiro acessa a página de consultas de viagens.
  2. O passageiro aplica filtros (data, destino, etc.).
  3. O sistema exibe as viagens disponíveis de acordo com os filtros.
  4. O passageiro pode selecionar uma viagem para mais informações.

Requisitos Especiais

  • O sistema deve ser capaz de aplicar múltiplos filtros de pesquisa.

Cenários Alternativos

  • Se não houver viagens disponíveis, o sistema exibe uma mensagem informando a indisponibilidade de viagens para os critérios selecionados.
