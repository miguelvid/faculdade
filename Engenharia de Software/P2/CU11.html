CU11: Gerenciar Usuários do Sistema

Identificador: CU11

Título: Gerenciar Usuários do Sistema

Ator(es): Administrador

Descrição: Este caso de uso permite ao administrador gerenciar as contas dos usuários do sistema, como criar, editar ou excluir usuários.

Pré-condições

  • O administrador deve estar autenticado no sistema.

Pós-condições

  • O sistema atualiza a base de dados com as modificações feitas nas contas de usuário.

Fluxo de Eventos

  1. O administrador acessa a página de gerenciamento de usuários.
  2. O administrador escolhe a ação (criar, editar ou excluir) para um usuário.
  3. O administrador insere ou altera as informações do usuário.
  4. O sistema valida e confirma a ação.
  5. O sistema atualiza o banco de dados com as informações modificadas.

Requisitos Especiais

  • O sistema deve garantir que apenas administradores possam acessar essa funcionalidade.

Cenários Alternativos

  • Se as informações inseridas forem inválidas, o sistema exibe uma mensagem de erro e não executa a ação.
