CU18: Alterar Rota de Viagem

Identificador: CU18

Título: Alterar Rota de Viagem

Ator(es): Administrador

Descrição: Este caso de uso permite ao administrador alterar a rota de uma viagem previamente agendada, ajustando o trajeto, pontos de parada ou horário.

Pré-condições

  • O administrador deve estar autenticado no sistema.
  • A viagem deve estar agendada no sistema.

Pós-condições

  • O sistema atualiza a rota da viagem no banco de dados.

Fluxo de Eventos

  1. O administrador acessa a página de gerenciamento de viagens.
  2. O administrador seleciona a viagem a ser alterada.
  3. O administrador altera a rota da viagem.
  4. O sistema valida e confirma a alteração da rota.
  5. O sistema atualiza a viagem com a nova rota.

Requisitos Especiais

  • O sistema deve garantir que a alteração da rota não afete outros aspectos da viagem, como horários e veículos.

Cenários Alternativos

  • Se a rota alterada não for válida (ex: conflitante com outras viagens), o sistema exibe uma mensagem de erro.
