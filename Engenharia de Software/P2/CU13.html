CU13: Consultar Detalhes da Viagem

Identificador: CU13

Título: Consultar Detalhes da Viagem

Ator(es): Passageiro

Descrição: Este caso de uso permite ao passageiro consultar os detalhes de uma viagem, como horários, assentos e outras informações relevantes.

Pré-condições

  • O passageiro deve estar autenticado no sistema.
  • A viagem escolhida deve estar disponível para consulta.

Pós-condições

  • O sistema exibe todos os detalhes da viagem selecionada.

Fluxo de Eventos

  1. O passageiro acessa a página da viagem desejada.
  2. O sistema exibe informações como data, horário, assentos disponíveis e preços.
  3. O passageiro pode tomar ações adicionais, como comprar ou reservar um assento.

Requisitos Especiais

  • O sistema deve exibir todos os detalhes de forma clara e objetiva.

Cenários Alternativos

  • Se não houver informações disponíveis sobre a viagem, o sistema exibe uma mensagem informando a ausência de detalhes.
