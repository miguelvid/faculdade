CU08: Verificar Disponibilidade de Assentos

Identificador: CU08

Título: Verificar Disponibilidade de Assentos

Ator(es): Passageiro

Descrição: Este caso de uso permite ao passageiro verificar a disponibilidade de assentos para uma determinada viagem antes de fazer a compra.

Pré-condições

  • O passageiro deve estar autenticado no sistema.
  • A viagem escolhida deve estar disponível para compra.

Pós-condições

  • O sistema exibe a disponibilidade de assentos para a viagem escolhida.

Fluxo de Eventos

  1. O passageiro acessa a página da viagem desejada.
  2. O sistema exibe a lista de assentos disponíveis para seleção.
  3. O passageiro escolhe um assento.
  4. O sistema confirma a disponibilidade do assento e permite a compra.

Requisitos Especiais

  • O sistema deve exibir a disponibilidade de assentos em tempo real.
  • O sistema deve garantir que a escolha do assento seja refletida no processo de compra.

Cenários Alternativos

  • Se o assento escolhido já estiver ocupado, o sistema informa ao passageiro e solicita uma nova escolha.
