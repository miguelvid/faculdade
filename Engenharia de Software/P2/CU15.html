CU15: Consultar Disponibilidade de Veículos

Identificador: CU15

Título: Consultar Disponibilidade de Veículos

Ator(es): Administrador

Descrição: Este caso de uso permite ao administrador verificar a disponibilidade dos veículos para viagens, considerando datas e horários.

Pré-condições

  • O administrador deve estar autenticado no sistema.

Pós-condições

  • O sistema exibe a disponibilidade dos veículos de acordo com as viagens programadas.

Fluxo de Eventos

  1. O administrador acessa a página de consulta de veículos.
  2. O administrador aplica os filtros (data, horário, destino).
  3. O sistema exibe a lista de veículos disponíveis.
  4. O administrador pode alocar ou realocar veículos conforme necessário.

Requisitos Especiais

  • O sistema deve garantir a exibição precisa das informações de disponibilidade em tempo real.

Cenários Alternativos

  • Se não houver veículos disponíveis para os critérios escolhidos, o sistema exibe uma mensagem informando a indisponibilidade.
