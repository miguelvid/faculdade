CU19: Excluir Viagem Agendada

Identificador: CU19

Título: Excluir Viagem Agendada

Ator(es): Administrador

Descrição: Este caso de uso permite ao administrador excluir uma viagem previamente agendada no sistema, removendo-a completamente.

Pré-condições

  • O administrador deve estar autenticado no sistema.
  • A viagem deve estar agendada e não ter sido realizada ainda.

Pós-condições

  • O sistema exclui a viagem do banco de dados.

Fluxo de Eventos

  1. O administrador acessa a página de gerenciamento de viagens.
  2. O administrador seleciona a viagem a ser excluída.
  3. O administrador confirma a exclusão da viagem.
  4. O sistema valida e exclui a viagem do banco de dados.

Requisitos Especiais

  • O sistema deve garantir que apenas administradores possam excluir viagens.

Cenários Alternativos

  • Se a viagem já foi realizada ou está em andamento, o sistema exibe uma mensagem de erro e não permite a exclusão.
