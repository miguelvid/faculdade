CU20: Atribuir Motorista a Veículo

Identificador: CU20

Título: Atribuir Motorista a Veículo

Ator(es): Administrador

Descrição: Este caso de uso permite ao administrador atribuir um motorista a um veículo específico para uma viagem.

Pré-condições

  • O administrador deve estar autenticado no sistema.
  • O motorista deve estar registrado no sistema e disponível.

Pós-condições

  • O sistema atualiza o veículo com a informação do motorista designado.

Fluxo de Eventos

  1. O administrador acessa a página de gerenciamento de veículos.
  2. O administrador seleciona o veículo a ser configurado.
  3. O administrador escolhe o motorista disponível.
  4. O sistema valida e confirma a atribuição do motorista ao veículo.
  5. O sistema atualiza o banco de dados com a nova atribuição.

Requisitos Especiais

  • O sistema deve garantir que o motorista atribuído esteja disponível para a viagem.

Cenários Alternativos

  • Se o motorista escolhido não estiver disponível, o sistema exibe uma mensagem de erro.
