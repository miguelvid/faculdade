CU05: Consultar Status do Pagamento

Identificador: CU05

Título: Consultar Status do Pagamento

Ator(es): Passageiro

Descrição: Este caso de uso permite ao passageiro verificar o status do pagamento de uma passagem previamente adquirida.

Pré-condições

  • O passageiro deve estar autenticado no sistema.
  • A passagem deve ter sido comprada previamente.

Pós-condições

  • O sistema exibe o status atual do pagamento (aprovado, pendente, recusado).

Fluxo de Eventos

  1. O passageiro acessa a página de "Minhas Passagens".
  2. O passageiro seleciona a passagem desejada.
  3. O sistema exibe o status atual do pagamento.
  4. O passageiro pode tomar uma ação dependendo do status (por exemplo, tentar um novo pagamento se for recusado).

Requisitos Especiais

  • O sistema deve garantir a precisão das informações exibidas sobre o status do pagamento.

Cenários Alternativos

  • Se o pagamento for recusado, o sistema exibe uma mensagem explicando o motivo e permite que o passageiro tente novamente.
