CU21: Consultar Veículo Disponível

Identificador: CU21

Título: Consultar Veículo Disponível

Ator(es): Administrador

Descrição: Este caso de uso permite ao administrador consultar os veículos disponíveis para uma viagem ou transporte, verificando os que estão livres e aptos para operar.

Pré-condições

  • O administrador deve estar autenticado no sistema.

Pós-condições

  • O sistema exibe os veículos disponíveis para a viagem ou tarefa solicitada.

Fluxo de Eventos

  1. O administrador acessa a página de consulta de veículos.
  2. O administrador consulta os veículos disponíveis.
  3. O sistema exibe a lista de veículos livres para uso.
  4. O administrador seleciona o veículo desejado para a viagem.

Requisitos Especiais

  • O sistema deve garantir que a disponibilidade do veículo esteja atualizada em tempo real.

Cenários Alternativos

  • Se não houver veículos disponíveis, o sistema exibe uma mensagem de erro.
