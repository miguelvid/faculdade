CU12: Gerenciar Frota

Identificador: CU12

Título: Gerenciar Frota

Ator(es): Administrador

Descrição: Este caso de uso permite ao administrador adicionar, editar ou remover veículos da frota.

Pré-condições

  • O administrador deve estar autenticado no sistema.

Pós-condições

  • O sistema atualiza as informações da frota com base nas alterações feitas.

Fluxo de Eventos

  1. O administrador acessa a página de gerenciamento de frota.
  2. O administrador escolhe a opção de adicionar, editar ou remover um veículo.
  3. O administrador insere ou modifica os dados do veículo.
  4. O sistema valida as informações inseridas.
  5. O sistema atualiza a frota com os dados fornecidos.

Requisitos Especiais

  • O sistema deve garantir que apenas administradores tenham acesso a essa funcionalidade.
  • O sistema deve validar os dados inseridos, como número de placa e capacidade do veículo.

Cenários Alternativos

  • Se as informações fornecidas forem inválidas, o sistema exibe uma mensagem de erro.
