CU02: Cancelar Passagem

Identificador: CU02

Título: Cancelar Passagem

Ator(es): Passageiro

Descrição: Este caso de uso permite ao passageiro cancelar uma passagem previamente comprada, com base nas regras de política de cancelamento da empresa.

Pré-condições

  • O passageiro deve estar autenticado no sistema.
  • A passagem a ser cancelada deve ser válida (dentro do prazo de cancelamento).

Pós-condições

  • A passagem é cancelada no sistema.
  • O reembolso (se aplicável) é processado.

Fluxo de Eventos

  1. O passageiro acessa a página "Minhas Passagens".
  2. O passageiro escolhe a passagem que deseja cancelar.
  3. O sistema verifica se o cancelamento é permitido.
  4. O passageiro confirma o cancelamento.
  5. O sistema processa o cancelamento e reembolso, se aplicável.
  6. O passageiro recebe uma notificação de cancelamento.

Requisitos Especiais

  • O sistema deve validar a política de cancelamento antes de proceder.
  • O sistema deve garantir que o reembolso seja processado de acordo com as regras financeiras da empresa.

Cenários Alternativos

  • Se a política de cancelamento não permitir o cancelamento, o sistema exibe uma mensagem informando o motivo.
