CU22: Excluir Reclamação

Identificador: CU22

Título: Excluir Reclamação

Ator(es): Administrador

Descrição: Este caso de uso permite ao administrador excluir uma reclamação que não é válida ou que foi resolvida de maneira satisfatória.

Pré-condições

  • O administrador deve estar autenticado no sistema.
  • A reclamação deve estar registrada no sistema.

Pós-condições

  • O sistema exclui a reclamação do banco de dados.

Fluxo de Eventos

  1. O administrador acessa a página de reclamações registradas.
  2. O administrador seleciona a reclamação a ser excluída.
  3. O administrador confirma a exclusão.
  4. O sistema valida e exclui a reclamação.

Requisitos Especiais

  • O sistema deve garantir que a exclusão da reclamação seja irreversível após a confirmação do administrador.

Cenários Alternativos

  • Se a reclamação estiver em andamento ou sendo analisada, o sistema exibe uma mensagem informando que não pode ser excluída.
