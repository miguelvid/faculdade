CU09: Registrar Reclamação

Identificador: CU09

Título: Registrar Reclamação

Ator(es): Passageiro

Descrição: Este caso de uso permite ao passageiro registrar uma reclamação sobre um serviço ou viagem, como problemas com o veículo, atraso, ou comportamento de funcionários.

Pré-condições

  • O passageiro deve estar autenticado no sistema.

Pós-condições

  • O sistema cria uma reclamação no banco de dados para que seja analisada pela equipe responsável.

Fluxo de Eventos

  1. O passageiro acessa a página de reclamações.
  2. O passageiro preenche o formulário com as informações da reclamação.
  3. O sistema valida as informações inseridas e registra a reclamação.
  4. O sistema envia uma confirmação de que a reclamação foi registrada.

Requisitos Especiais

  • O sistema deve garantir que a reclamação seja registrada corretamente e enviada para o departamento responsável.

Cenários Alternativos

  • Se as informações forem inválidas ou incompletas, o sistema solicita que o passageiro corrija antes de enviar a reclamação.
