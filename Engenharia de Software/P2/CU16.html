CU16: Atualizar Status do Veículo

Identificador: CU16

Título: Atualizar Status do Veículo

Ator(es): Administrador

Descrição: Este caso de uso permite ao administrador atualizar o status de um veículo, como se está em manutenção, disponível ou fora de operação.

Pré-condições

  • O administrador deve estar autenticado no sistema.

Pós-condições

  • O status do veículo é atualizado no sistema.

Fluxo de Eventos

  1. O administrador acessa a página de gerenciamento de veículos.
  2. O administrador escolhe o veículo que deseja atualizar.
  3. O administrador altera o status do veículo.
  4. O sistema valida e confirma a alteração.
  5. O sistema atualiza o banco de dados com o novo status.

Requisitos Especiais

  • O sistema deve garantir que apenas administradores possam acessar essa funcionalidade.

Cenários Alternativos

  • Se o status do veículo não for válido (exemplo: status inconsistente), o sistema exibe uma mensagem de erro.
