CU01: Comprar Passagem Online

Identificador: CU01

Título: Comprar Passagem Online

Ator(es): Passageiro

Descrição: Este caso de uso descreve o processo de compra de uma passagem para uma viagem selecionada.

Pré-condições

  • O passageiro deve estar autenticado no sistema.
  • O passageiro deve ter acesso a uma forma de pagamento válida.
  • Existem viagens disponíveis para compra.

Pós-condições

  • A passagem é gerada e enviada ao passageiro via e-mail.
  • O pagamento é processado e confirmado.

Fluxo de Eventos

  1. O passageiro acessa a página de viagens disponíveis.
  2. O passageiro seleciona a viagem desejada.
  3. O passageiro escolhe o assento e informa os dados de pagamento.
  4. O sistema valida o pagamento.
  5. O sistema confirma a compra e envia a passagem para o e-mail do passageiro.
  6. O passageiro é redirecionado à página de confirmação.

Cenários Alternativos

  • Se o pagamento for recusado, o sistema exibe uma mensagem de erro e solicita um novo método de pagamento.
