CU23: Visualizar Detalhes de Reclamação

Identificador: CU23

Título: Visualizar Detalhes de Reclamação

Ator(es): Administrador, Passageiro

Descrição: Este caso de uso permite ao administrador ou passageiro visualizar os detalhes de uma reclamação registrada, para acompanhar o andamento e resolução do problema.

Pré-condições

  • O passageiro ou administrador deve estar autenticado no sistema.

Pós-condições

  • O sistema exibe os detalhes da reclamação.

Fluxo de Eventos

  1. O passageiro ou administrador acessa a página de "Minhas Reclamações" ou "Reclamações Pendentes".
  2. O usuário seleciona a reclamação que deseja visualizar.
  3. O sistema exibe os detalhes da reclamação, incluindo data, descrição e status.
  4. O administrador pode marcar a reclamação como resolvida, se for o caso.

Requisitos Especiais

  • O sistema deve garantir que apenas administradores ou o próprio passageiro possam visualizar as reclamações específicas.

Cenários Alternativos

  • Se a reclamação não existir ou não for encontrada, o sistema exibe uma mensagem de erro.
