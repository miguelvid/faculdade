CU04: Realizar Pagamento

Identificador: CU04

Título: Realizar Pagamento

Ator(es): Passageiro

Descrição: Este caso de uso permite que o passageiro faça o pagamento da passagem utilizando um método de pagamento válido.

Pré-condições

  • O passageiro deve estar autenticado no sistema.
  • O passageiro deve ter selecionado uma passagem para compra.

Pós-condições

  • O pagamento é processado e a passagem é confirmada.

Fluxo de Eventos

  1. O passageiro acessa a página de pagamento.
  2. O passageiro insere as informações de pagamento (cartão, PayPal, etc.).
  3. O sistema valida as informações de pagamento.
  4. O pagamento é processado e o sistema confirma a compra.

Requisitos Especiais

  • O sistema deve ser capaz de aceitar vários tipos de pagamento.
  • O sistema deve garantir a segurança das transações financeiras.

Cenários Alternativos

  • Se o pagamento for recusado, o sistema exibe uma mensagem de erro e solicita um novo método de pagamento.
