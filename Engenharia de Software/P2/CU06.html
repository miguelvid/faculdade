CU06: Consultar Histórico de Compras

Identificador: CU06

Título: Consultar Histórico de Compras

Ator(es): Passageiro

Descrição: Este caso de uso permite ao passageiro consultar as compras passadas de passagens realizadas no sistema.

Pré-condições

  • O passageiro deve estar autenticado no sistema.

Pós-condições

  • O sistema exibe o histórico de compras de passagens.

Fluxo de Eventos

  1. O passageiro acessa a página "Histórico de Compras".
  2. O sistema exibe uma lista de compras anteriores, com detalhes das passagens adquiridas.
  3. O passageiro pode selecionar uma compra para ver mais detalhes.

Requisitos Especiais

  • O sistema deve garantir a exibição de informações completas sobre cada compra realizada.

Cenários Alternativos

  • Se não houver compras realizadas, o sistema exibe uma mensagem informando que não há registros.
