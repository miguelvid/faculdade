CU10: Verificar Histórico de Viagens

Identificador: CU10

Título: Verificar Histórico de Viagens

Ator(es): Passageiro

Descrição: Este caso de uso permite ao passageiro consultar o histórico das viagens realizadas, incluindo detalhes sobre datas, horários e status de pagamento.

Pré-condições

  • O passageiro deve estar autenticado no sistema.

Pós-condições

  • O sistema exibe uma lista com o histórico de viagens do passageiro.

Fluxo de Eventos

  1. O passageiro acessa a página de "Minhas Viagens".
  2. O sistema exibe a lista de viagens realizadas.
  3. O passageiro pode selecionar uma viagem para visualizar mais detalhes.

Requisitos Especiais

  • O sistema deve garantir que o histórico de viagens seja acessado de forma segura e precisa.

Cenários Alternativos

  • Se não houver viagens registradas, o sistema exibe uma mensagem informando a ausência de histórico.
